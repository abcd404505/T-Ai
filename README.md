# ⚡ T-AI: Advance Terminal AI for CLI-OS

A lightweight and fast interactive Terminal AI Tool designed for Termux and Linux CLI. Powered by advanced LLM engines with a cool 3D banner and threaded loading animation.

---

## ✨ Features

- **Multi-Engine:** Choose between `gpt-5`, `claude-opus-4.7`, and `gpt-4o`.
  
- **Elegant UI:** Stylized ANSI color codes and 3D status banner.

- **Asynchronous Loading:** Smooth, background-threaded `Thinking...` animation.

- **Auto-Installer:** Automatically installs missing dependencies (`requests`).

---

## 🚀 Quick Start (Termux/Linux)

```bash
pkg update && pkg upgrade -y
pkg install python git -y
git clone [https://github.com/YOUR_USERNAME/YOUR_REPO.git](https://github.com/YOUR_USERNAME/YOUR_REPO.git)
cd YOUR_REPO
python ai_tool.py
```

## ⚙️ Commands
**​clear** - Clear the terminal and refresh the banner.
**​exit / quit** - Close the tool safely.
